# Contributing to JobSpark AI

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/yourusername/jobspark-ai.git`
3. Create a branch: `git checkout -b feature/your-feature`
4. Make changes and commit: `git commit -m "Add your feature"`
5. Push to your fork: `git push origin feature/your-feature`
6. Create a Pull Request

## Development Setup

1. Run setup script: `scripts\setup.bat`
2. Start development: `scripts\start_all.bat`

## Code Standards

- Follow existing code style
- Add tests for new features
- Update documentation
- Ensure all tests pass

## Reporting Issues

Use GitHub Issues to report bugs or request features.
